# Adaptive MCQ Generation using Bloom’s Taxonomy

## Overview
This project explores an AI-driven approach to generating and adapting multiple-choice questions (MCQs) based on Bloom’s Taxonomy and real-time student performance.

## Problem
Traditional assessments are static and fail to adapt to individual learning levels. This leads to inefficient learning and poor evaluation of conceptual understanding.

## Approach
The system is designed as a closed-loop pipeline:

1. Input: Topic / Concept
2. Question Generation using LLMs (aligned with Bloom’s levels: Remember, Understand, Apply, etc.)
3. Student Response Evaluation
4. Performance Tracking (accuracy, response time, consistency)
5. Adaptive Difficulty Adjustment
6. Feedback loop for next question generation

## Key Ideas
- Cognitive-level-aware question generation
- Real-time adaptation using student performance signals
- Prompt engineering for controlling difficulty and depth
- Evaluation strategies for LLM-generated questions

## Status
Currently in research and system design phase. Implementation in progress.
